# 通用模块

# 获取当前学期和时间

# 路径

/api/time GET

# 参数

无

# 响应

  • term：当前学期，例：2019/2020(1)
  • month：当前月份（1-12）
  • day：当前日（1-7）
  • week：当前周次（0+）
  • is_begin：学期是否开始（周次 > 0）

# 状态

错误码 状态
1 获取时间成功

# 例子

{
  "errcode": 1,
  "errmsg": "获取时间成功",
  "data": {
    "term": "2019/2020(1)",
    "month": "12",
    "week": 13,
    "day": 6,
    "is_begin": true
  },
  "redirect": null
}

# 绑定正方教务账号

# 路径

/api/zf/bind POST 需要认证

# 备注

该 API 修改微精弘服务端记录的正方教务密码，并接数据服务器进行验证。

# 参数

  • password：正方教务密码

# 响应

用户数据对象

# 状态

错误码 状态
1 绑定正方账号成功
-1 没有认证信息
-1 用户名或密码错误\nps:正方密码是你选课的密码
-1 其他错误

# 例子

{
  "errcode": 1,
  "errmsg": "绑定正方账号成功",
  "data": {
    "id": "id",
    "uno": "uno",
    "...": "..."
  },
  "redirect": null
}
