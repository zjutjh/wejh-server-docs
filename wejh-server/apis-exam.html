# 排考模块

# 获取排考信息

# 路径

/api/ycjw/exam GET 需要认证

# 参数

无

# 响应

  • list：排考信息
  • term：学期，例：2019/2020(1)

# list

  • 班级：班级
  • 教师：教师
  • 课程：课程
  • 日期：日期
  • 时段：时段，正方仅返回（实际为准）
  • 教室：教室
  • 座位号：座位号 仅正方系统
  • 考试方式：考试方式 / 暂无考试方式 仅正方系统
  • 课程名称：课程名称 仅正方系统
  • 日、星期、星期名、倒计时 和 倒计时名 的值难以确定其格式，不建议客户端使用。

# 状态

错误码 状态
1 获取排考成功
-1 没有认证信息
-1 需要绑定（用户名或密码为空）
-1 请重新绑定正方账号（用户名或密码错误）
-1 其他错误

# 学期切换

# 路径

/api/Ycjw/exam PUT 需要认证

# 备注

该 API 仅修改微精弘服务端中的学期信息，不连接数据服务器进行查询。

# 参数

  • term：学期，例：2019/2020(1)

# 响应

空

# 状态

错误码 状态
1 切换学期成功
-1 没有认证信息

# 例子

{
  "errcode": 1,
  "errmsg": "切换学期成功",
  "data": null,
  "redirect": null
}
